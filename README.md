# TaskNest

TaskNest is a Java console-based freelance task management system.

## Features

- Client Login
- Freelancer Login
- Create Task
- Update Task
- View Tasks
- Apply for Task
- View Applicants
- Assign Freelancer
- Budget Sorting
- Exception Handling

## Technologies Used

- Java
- OOP
- Collections
- Exception Handling
- IntelliJ IDEA
